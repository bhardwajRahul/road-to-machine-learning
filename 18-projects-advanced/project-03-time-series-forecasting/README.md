# Project 3: Time Series Forecasting (Advanced)

Build advanced forecasting models for complex time series data.

> **Pick one path:** Overlaps with [Module 15](../../15-time-series-analysis/README.md) and [intermediate project 6](../../17-projects-intermediate/project-06-time-series-forecasting/README.md). See [TIME_SERIES_LEARNING_PATH.md](../../TIME_SERIES_LEARNING_PATH.md).

## Difficulty
Advanced

## Time Estimate
1-2 weeks

## Skills You'll Practice
- Time Series Analysis
- LSTM
- ARIMA
- Prophet

## Learning Objectives

- Apply advanced time series techniques
- Use LSTM for sequences
- Implement Prophet for trends
- Engineer features for time series
- Multi-step ahead forecasting

## Dataset

Stock prices, energy consumption, or sales data

## Project Steps

1. Load and analyze time series
2. Implement ARIMA
3. Build LSTM model
4. Use Prophet
5. Compare methods
6. Multi-step forecasting
